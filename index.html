Let's get our Cheet On!

Terminal Commands

cd Current Directory
cp Copy
chmod Change File Mode Bits
ls List Directory Contents
mkdir Make Directories
move Move Files
pwd Print Working Directory
rm Remove Files

Configure

git config --global user.name "[name]"
git config --global user.email "[email address]"
git config --global color.ui aut

Saving

git status List changed and new files that till has to be commited.
git add [file] Adding file to list of files that still need to be commited.
git commit -m "[descriptive message]" The file is permanently saved.

Clone

git clone [url]

Upload and Download

git push [alias] [branch] Upload commits
git pull Downloads History
