Sign Up
  1. Download our App (Scroll down for the link to the App)
  2. Open App
  3. Select the "Sign-up" option
  4. Enter credentials
  5. Select "Submit"

Your application may take up to 48h to be processed. Thank you for your patience.

Make Payment
  1. Open App
  2. Log on using your own credentials
  3. Select "Manage Account"
  4. Select "Payments"
  5. Select ""
Change Purchase Limit
  1. Open App
  2. Log on using your own credentials
  3. Select "Manage Account"
  4. Select "Edit Purchase Limit"
  5. Enter the amount which should be your new limit
  6. Select "Update"
Scheduled Payments
  1. Open App
  2. Log on using your own credentials
  3. Select "Manage Account"
  4. Select "Scheduled Payments"
Transaction History
  1. Open App
  2. Log on using your own credentials
  3. Select "Manage Account"
  4. Select "Transactions"
Activate Card
  1. Open App
  2. Log on using your own credentials
  3. Select "Manage Account"
  4. Select "Cards"
  5. Select "Activate Card"
  6. Enter card credentials
  7. Select "Activate"

Deactivate Card
  1. Open App
  2. Log on using your own credentials
  3. Select "Manage Account"
  4. Select "Cards"
  5. Select the account card that you wish to deactivate
  6. Select "Deactivate Card"
Beneficiaries
  1. Open App
  2. Log on using your own credentials
  3. Select "Manage Account"
  4. Select "Beneficiaries"
